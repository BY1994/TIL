### gerrit에 tag 푸시

gerrit은 annotated tag (주석 달린 tag) 만 반영할 수 있다고 한다.

```shell
git checkout v2.0
git tag -am "Adding v2.0.1 tag" v2.0.1
git push origin v2.0.1 HEAD:refs/heads/v2.0
```
