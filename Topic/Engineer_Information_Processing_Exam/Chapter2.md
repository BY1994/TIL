- 중위 순회 : 왼쪽 서브트리 -> 중간 노드 -> 오른쪽 서브트리
- 전위 순회 : 중간 노드 -> 왼쪽 서브트리 -> 오른쪽 서브트리
- 후위 순회 : 왼쪽 서브트리 -> 오른쪽 서브트리 -> 중간 노드

- 개념 스키마
- 내부 스키마

- 요구조건 분석
